# Rails Quest

Учебный проект для практики разработки на Ruby on Rails.
Стек: Ruby on Rails, esbuild, Tailwind CSS, SQLite.

## Квесты

### Квест 1. Работа с базой данных

Миграции, модели, ассоциации, внешние ключи и валидации.
Студент по шагам восстанавливает предметную область: агенты, навыки и задания.

### Квест 2. Запросы к базе данных
По уже восстановленной базе данных нужно построить выборки с помощью ActiveRecord.


## Запуск

```bash
bin/setup
bin/dev
```

## Полезные команды

Прогнать тесты выбранного квеста:

```bash
bin/rails quest:check[1]
```
Принять квест (должны быть успешно пройдены все тесты):
```bash
bin/rails quest:accept[1]
```
